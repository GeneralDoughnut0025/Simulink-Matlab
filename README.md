# Simulink-Matlab

This repository contains my MATLAB and Simulink projects focused on modeling, simulation, analysis, and visualization of engineering and energy systems.

The work includes projects such as:
- Automatic Voltage Regulator (AVR)
- Solar power and panel-to-grid systems
- Wind power plant modeling
- Battery charger simulation
- Multilevel inverter modeling
- Three-phase fault analysis
- Long transmission line modeling
- Graphs, plots, and MATLAB scripting practice

For a quick overview of my MATLAB and Simulink work, please check [SIMULINK_MATLAB_PROOF.pdf](./SIMULINK_MATLAB_PROOF.pdf).
